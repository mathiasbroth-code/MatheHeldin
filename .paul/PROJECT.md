# Project: MatheHeldin

## Description
Eine App (Handy, Tablet, PC) in der Kinder Mathematik parallel zum Schulunterricht lernen, üben und die Materie verstehen sollen. Es soll ihnen Spaß machen, weil sie auch sehen, wie sie besser werden.

## Core Value
Philippa kann selbstständig Mathe üben und ihre Eltern sehen den Lernfortschritt. Das Programm hilft ihr, die Materie von Anfang an zu verstehen — mit pädagogisch sinnvollen Lerntipps, die SIE verstehen und umsetzen kann.
