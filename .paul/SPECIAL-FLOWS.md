## Design System (Richtung 3 — Clean Minimal)

Verbindliche Gestaltungsrichtlinie für alle UI-Arbeit:

**Farben:**
- Background: Weiß `#ffffff`
- Cards: `#f8fafc`
- Borders: `#e2e8f0`
- Primary (Teal): `#0d9488`
- Erfolg: `#10b981`
- Warnung: `#f59e0b`
- Text Überschriften: `#0f172a`
- Text Fließtext: `#475569`
- Text Sekundär: `#64748b`

**Typografie:**
- Font: Inter / System-Stack
- Zahlen: `font-variant-numeric: tabular-nums`
- Überschriften: Inter semibold
- Fließtext: Inter regular

**Komponenten:**
- Karten: `rounded-2xl`, kein box-shadow, `1px solid #e2e8f0`
- Buttons Primary: Teal solid
- Buttons Secondary: Ghost (Teal border + text)
- Inputs: 2px Border grau, Focus = Teal border + 3px Ring
- Feedback Richtig: `#f0fdf4` BG
- Feedback Falsch: `#fffbeb` BG + "🤔 Noch nicht ganz"
- Progress: 8px hoch, Track `#e2e8f0`, Fill `#14b8a6`
- Stage-Karten: Card + Border, Icon in farbigem Kreis, Titel bold, Sub grau, Mini-Progressbar

**Dienes-Farben:** unverändert (Einer hellgelb → Tausender orange)
