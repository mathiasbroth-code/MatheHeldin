<objective>
## Goal
Port the 4 "Lesen" stages (mengen20, bis100lesen, bis1000lesen, bis10000lesen) using a shared ZahlLesenView component. All show Dienes BlockDisplay and ask "Welche Zahl ist das?" — only the number range differs.

## Purpose
These 4 stages cover the core "Zahl erkennen" skill from Einern bis Tausendern. Sharing one View component keeps code DRY and validates that the Stage-Pattern works with parametrized shared Views.

## Output
- `src/stages/shared/ZahlLesenView.tsx` — shared View for all Lesen-Stages
- 4 stage modules: mengen20, bis100lesen, bis1000lesen, bis10000lesen
- All 4 registered in registry.ts (total: 5 stages including mitte)
</objective>
