<objective>
## Goal
Create a profile selection start screen so multiple children can use the app. Each child picks their avatar and name, then proceeds to the stage overview. Replace the current auto-profile hack with a proper profile system.

## Purpose
Without profiles, all progress is mixed. The Konzept requires each child to have their own profile with avatar and name, so parents can see who practiced what. This is the foundation for the Eltern-Dashboard (Phase 6).

## Output
- `/` = ProfilAuswahl (pick who's practicing)
- `/profil/neu` = ProfilErstellen (new profile)
- `/ueben` = Stufen-Übersicht (moved from /)
- `/stufe/:id` = unchanged
- Profile schema with avatar field (Dexie migration v2)
- Auto-profile hack in StufeView removed

## Phase 3 Overview (2 Plans)
- **03-01**: Profil-Auswahl + Schema + Route-Umbau (THIS PLAN)
- **03-02**: Tab-Navigation + Fortschritt-Seite + Eltern-PIN
</objective>
