<objective>
## Goal
Build the 3-level tip infrastructure and session summary. After this plan, any stage can plug in tips and explanations. The `mitte` stage (which already has a basic tip) serves as the integration test.

## Purpose
Tips are the core pedagogical feature — they help children understand, not just guess. The 3-level scaffolding (Denkanstoß → Methode → Schritt-für-Schritt) lets each child decide how much help they need. Session summaries celebrate progress.

## Output
- Dexie schema v3 with `tippStufe` in AntwortLog
- Extended Stage<T> interface with `erklaerung` and `tipps`
- `useTipp` hook managing 3-level state
- `TippSystem` component (replaces simple TippButton)
- `ErklaerungButton` component ("Was lernst du hier?")
- `SessionSummary` component (shown after reaching zielRichtige)
- StufeView integration: passes tippStufe to recorder, shows summary
- `mitte` stage updated as proof-of-concept with all 3 tip levels + erklaerung

## Phase 5 Overview (2 Plans)
- **05-01**: Infrastructure + mitte integration (THIS PLAN)
- **05-02**: Add tipps + erklaerung content to remaining 10 stages
</objective>
