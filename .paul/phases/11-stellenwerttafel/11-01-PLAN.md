<objective>
## Goal
Stellenwerttafel-Visualisierung für Aufgaben die Zahlen in Stellenwerte zerlegen
(M|HT|ZT|T|H|Z|E). Erkennt Zahlen im Aufgabentext und zeigt sie visuell aufgeschlüsselt.

## Purpose
Philippa sieht sofort, welche Ziffer an welcher Stelle steht — das ist die Basis
für Zahlenverständnis bis zur Million. Das Buch nutzt Stellenwerttafeln auf S. 18-22.

## Output
- StellenwertTafel.tsx — Visualisierungs-Komponente
- Integration in AufgabeWrapper (kaskadiert, wie die anderen Visualisierungen)
</objective>

<acceptance_criteria>

## AC-1: Stellenwerttafel zeigt Zahl korrekt zerlegt
```gherkin
Given eine Aufgabe mit "Stellenwerttafel" oder "Stellenwert" im Text
When die Aufgabe eine Zahl enthält (z.B. 352.000)
Then wird eine Tafel mit Spalten M|HT|ZT|T|H|Z|E angezeigt
And jede Ziffer steht in der richtigen Spalte
And leere Stellen zeigen "0" oder bleiben leer
```

## AC-2: Erkennung greift nur bei relevanten Aufgaben
```gherkin
Given eine Aufgabe ohne Stellenwert-Bezug
When die Aufgabe gerendert wird
Then wird keine Stellenwerttafel angezeigt
```

```gherkin
Given alle Änderungen implementiert
When tsc --noEmit und pnpm build ausgeführt werden
Then gibt es keine Fehler
```
